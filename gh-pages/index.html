fxbuild

FiveM Resource Builder + Obfuscator - Um comando para buildar e ofuscar seus recursos FiveM.

Uso

curl -sL https://developerstriker.github.io/lua-obfuscator/install.sh | bash -s -- recurso/fxmanifest.lua --preset Weak

Exemplos

Minify (sem ofuscação):
curl -sL https://developerstriker.github.io/lua-obfuscator/install.sh | bash -s -- fxmanifest.lua --preset Minify
Ofuscação fraca:
curl -sL https://developerstriker.github.io/lua-obfuscator/install.sh | bash -s -- fxmanifest.lua --preset Weak
Ofuscação média:
curl -sL https://developerstriker.github.io/lua-obfuscator/install.sh | bash -s -- fxmanifest.lua --preset Medium
Ofuscação forte:
curl -sL https://developerstriker.github.io/lua-obfuscator/install.sh | bash -s -- fxmanifest.lua --preset Strong

Presets disponíveis

  • Minify - Apenas minifica (sem ofuscação)
  • Weak - Ofuscação leve
  • Medium - Ofuscação média
  • Strong - Ofuscação forte
